Features

Built for security teams and the engineers they trust.

Everything your CISO needs. Nothing that slows your developers down.

⊳
Sub-millisecond detection
Rule layers respond in under 1ms. Only genuine edge cases reach the LLM judge — latency invisible, costs minimal.
⊘
PII input and output protection
Detect SSNs, credit cards, emails, API keys in both prompts and LLM responses. Stop data from entering or leaving your AI pipeline.
⟳
Self-learning detection
Every threat caught by the LLM judge is saved as a learned pattern. The system improves with each scan — fewer expensive API calls over time.
≋
Real-time analytics dashboard
Live threat feed, severity charts, scan history, block rates, and confidence scores. Full visibility into your AI security posture.
⊞
Instant webhook alerts
Get Slack or webhook notifications the moment HIGH or CRITICAL threats are detected. Your security team knows before your users do.
≡
Compliance-ready audit logs
Full scan history with timestamps, threat types, confidence scores, and layer data. Export for SOC-2, GDPR, and ISO reporting.
⌬
Per-key custom policies
Define different rules for different API keys. Internal dev tools can have relaxed rules while your public chatbot has strict enforcement.
⊕
MCP gateway with tool validation
Validate tool definitions before your agents call them. Catch description injection, dangerous schema fields, and untrusted MCP servers.
↗
One-line integration
Change one URL in your code. Works as a drop-in proxy for OpenAI, Claude, Gemini, and any LLM API. Zero infrastructure changes.
